💜 Nubank Landing Page Clone

Uma recriação moderna e responsiva da landing page do Nubank, desenvolvida com foco em UI/UX, animações suaves e interações elegantes utilizando apenas HTML, CSS e JavaScript puro.

🚀 Sobre o projeto

Esse projeto foi desenvolvido com o objetivo de aprimorar minhas habilidades em desenvolvimento front-end, explorando conceitos modernos de interface, responsividade e experiência do usuário.

A ideia não foi apenas “copiar um layout”, mas sim criar uma experiência visual moderna, fluida e agradável, inspirada no estilo elegante do Nubank.

✨ Funcionalidades

✔️ Layout moderno e responsivo

✔️ Hero section com imagem em destaque

✔️ Carrossel interativo com: arrastar com mouse
momentum/inércia suave
navegação por indicadores
animações elegantes
efeito reveal ao scroll

✔️ Hover effects sofisticados

✔️ Efeito de zoom nas imagens dos cards

✔️ Scrollbar customizada

✔️ Glassmorphism

✔️ Animações suaves com JavaScript

✔️ Responsividade para desktop, tablet e mobile

✔️ Interface inspirada em aplicações reais do mercado
